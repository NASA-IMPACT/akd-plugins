# Multi-Hop: One Loop

Indirect (multi-hop) discovery is a **user-gated, single recursive pass**.

- Indirect variables may be surfaced only as candidate terms and hypotheses for search — never
  as asserted causal or validated relationships.
- Each indirect inference path requires **explicit user approval** before any further search.
- Inform the user of the intended GCMD keyword mapping before executing.
- **Limited to one recursive loop.** If a single indirect pass yields no datasets, stop and
  report rather than looping further.

There is no literature/citation tool: indirect variables come from the agent's own scientific
reasoning, and every surfaced dataset must still come from CMR. See `../reasoning.md`.
