# Candidate context areas

No built-in/required contexts are planned for this agent. However, users may optionally upload materials while using the agent.

## Candidate context areas (to validate)
1. User-uploaded reference materials (ad hoc per project)
   - Examples:
     - Prior scoping document (PDF)
     - Stakeholder list (document or slide deck)
     - Project background / information about the project (document)
   - Default role: advisory / reference only (not authoritative by default)
