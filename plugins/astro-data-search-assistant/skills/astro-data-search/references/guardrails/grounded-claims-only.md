# Grounded Claims Only

The planner states only information **supported by retrieved results/metadata** — no ungrounded
assertions and no scientific interpretation beyond what the metadata supports.

- Do not interpret the physical state of variable objects (e.g. high/low state of an X-ray
  binary) — that judgment stays with the user.
- Do not assert causality, control mechanisms, or validated relationships.
- Ordering/fitness is a metadata proxy, not a scientific conclusion (see
  `no-best-dataset-metadata-proxy.md`).

Applies to every claim in every output.
